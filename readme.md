#### Appium 方案
1. 下载和安装nodejs
2. npm install appium -g
3. 下载安卓SDK/JAVA 8/安卓模拟器，并使用adb命令连接模拟器，不同模拟器有不同的端口和名称，这一部分的详情google一下吧
4. 附上一个文档https://www.jianshu.com/p/a154b8b353b0
5. 打开appium服务端
6. 在demo这里，npm install 安装依赖
7. 启动脚本，npm run start
